# StickBoxing

## How to run

### Backend
```
1. cd backend
2. npm install
3. node Server.js
```

### Frontend
It's up to you what webserver you would like to use but I used https://www.npmjs.com/package/local-web-server

#### http-server installed
```
1. cd frontend
2. ws
```

#### local-web-server not installed
```
1. npm install -g local-web-server
2. cd frontend
3. ws
```
